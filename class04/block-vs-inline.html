block elements

A block-level block-level element may contain most other block-level elements and all inline elements. Block-level elements occupy at least one full line when rendering. Example of block level elements are paragraphs, divs, headers, lists, etc.

inline elements

An inline-level inline element may only contain other inline elements and text. Inline elements are rendered as they appear in the document. Example of inline elements are links, images, and span.
