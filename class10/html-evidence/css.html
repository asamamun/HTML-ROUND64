HyperText Markup Language (HTML) is the main markup language for creating web pages and other infromation that an be displayed in a web browser.


HTML is written in the form of HTML, elements consisting of tags enclose in angle brackets (like <html>, within the web page content. HTML tags most commonly come in paris like <h1> and </h1>, although some tags represent empty elements an so are unpaired, for eample <img>. The first tag in a pair is the start tag, an the second tag is the end tag (they are also called opening tags an colsing tags). In between these tags web designers can add text, ruther tags, comments and other types of text-based content.
